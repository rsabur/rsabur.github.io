HeirBnb is a web app for the lux traveler in mind. Users can browse available luxury listings by name, view their description, book an upcoming stay and add a review for their stays. A user can also delete a booking and a review.

Programming Languages

Built with Rails 5 API for quick setup, PostgreSQL for multiple data types and vanilla JavaScript frontend for simplicity

Styling

Designed with HTML5, custom CSS, Bootstrap carousel for landing page and MapBox API
