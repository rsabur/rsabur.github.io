About MyStyle

MyStyle is a web app designed to act as a virtual fitting room for the sylista in mind. Users can update their closet by adding articles of clothing, trying on seperates and creating new looks. From a profile page, a user can also view, update and delete their own information, as well as, change their model. UX/UI-focused outfit styling app.

Programming Languages

MyStyle was programmed with Rails 5 API server for ease, PostgreSQL for durability and React.js frontend.

UI Styling

Utilized Bootstrap, Material-UI and Semantic-UI for component styling and custom CSS for closet and clothing fit
